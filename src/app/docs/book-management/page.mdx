<Step>If you haven't completed the installation refer to guide [here](/docs/installation).</Step>

<Step>
  Manually create the file and folder structure based on the `book-management.ts.json` template in
  the registry.
</Step>

<Step>
  Create necessary directories such as `routes/`, `controllers/`, `models/`, and `services/` inside
  your project structure.
</Step>

<Step>
  Copy the TypeScript code blocks from the template into appropriate files (e.g., `book.routes.ts`,
  `book.controller.ts`, `book.model.ts`).
</Step>

<Step>
  Ensure your project is set up with a TypeScript compiler and proper tsconfig.json configuration.
</Step>

<Step>
  Connect to your preferred database (like PostgreSQL with Prisma or MySQL) and adjust any model or
  service files as needed.
</Step>

<Step>
  Test all core operations such as adding, retrieving, updating, and deleting books via the
  corresponding API endpoints.
</Step>
