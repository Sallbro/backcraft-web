<Step>If you haven't completed the installation refer to guide [here](/docs/installation).</Step>

<Step>
  Manually create the file and folder structure based on the `user-management-mysql.js.json`
  template in the registry.
</Step>

<Step>
  Create corresponding folders like `routes/`, `controllers/`, `models/`, and `middlewares/` in your
  project root.
</Step>

<Step>
  Paste the route handlers, controller logic, validation, and database models from the template into
  their respective files.
</Step>

<Step>
  Update any config values such as MySQL connection details (host, user, password, database) in your
  `.env` or `config.js` file.
</Step>

<Step>
  Run the migration (if defined), start your server, and test your user management endpoints:
  register, login, update, delete, etc.
</Step>
