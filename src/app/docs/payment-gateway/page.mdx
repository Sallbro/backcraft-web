  <TabsTrigger value="manual">Manual</TabsTrigger>
</TabsList>

<TabsContent value="manual">

<Steps>

<Step>If you haven't completed the installation refer to guide [here](/docs/installation).</Step>

<Step>
  Manually create the file and folder structure based on the `stripe-integration-js` template in the
  registry.
</Step>

<Step>
  Create necessary directories such as `routes/` and your root `app.js` file inside your project
  structure.
</Step>

<Step>
  Copy the JavaScript code blocks from the template into appropriate files (`app.js`,
  `routes/stripe.js`).
</Step>

<Step>Create a `.env` file and add your Stripe credentials.</Step>

<Step>Install the required dependencies using:</Step>

<CodeBlock
  code={`npm install express dotenv stripe body-parser`}
  lang="bash"
  className="h-[52px]"
/>

<Step>
  Run the server and test the `/api/stripe/payment` endpoint for successful payment intent creation.
</Step>

</Steps>

</TabsContent>

</Tabs>
